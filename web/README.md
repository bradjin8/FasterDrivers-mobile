# [Faster Drivers Web App]()


## Preliquisites
- [Node.js](https://nodejs.org/en/)
- [Yarn](https://yarnpkg.com/)
- [Git](https://git-scm.com/)
- [React](https://reactjs.org/)

## Run the project

1. Install dependencies
```bash
yarn install
```
2. Run the project
```bash
yarn start
```
3. Open the project in your browser
```bash
http://localhost:3000/
```
